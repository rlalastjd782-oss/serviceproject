# Changelog

이 프로젝트는 Semantic Versioning(`MAJOR.MINOR.PATCH`)을 기준으로 관리합니다.

## v1.5.1 - 2026-05-26

- 기록/분석 목록 페이징 기본 표시 개수를 모바일 기준에 맞춰 20개에서 10개로 낮췄습니다.
- 사용자가 직접 20개, 50개로 바꾸는 선택지는 유지했습니다.
- PWA 캐시와 앱 버전을 `v1.5.1`로 갱신했습니다.

## v1.5.0 - 2026-05-26

- 운동 입력에 `kg/lb` 단위 선택을 추가하고, `lb` 입력값은 저장 시 kg로 자동 변환되게 했습니다.
- 모바일 운동 입력을 여러 세트를 한 번에 만드는 세트 빌더 중심으로 개선했습니다.
- 3/4/5세트 프리셋, 같은 무게 채우기, 단계별 증량 채우기, kg 저장 미리보기를 추가했습니다.
- 세트 빌더 UI와 LB 변환 저장 테스트를 추가했습니다.
- PWA 캐시와 앱 버전을 `v1.5.0`으로 갱신했습니다.

## v1.4.0 - 2026-05-26

- 오늘 화면 첫 영역을 대시보드형 히어로로 바꾸고 핵심 액션을 명확히 했습니다.
- 기록 검색 결과를 수치 중심 카드형 리스트로 개선했습니다.
- 더보기 화면을 보조 도구 대시보드 톤으로 정리했습니다.
- 공통 카드/버튼/배경/빈 상태 스타일을 고도화했습니다.
- PWA 캐시와 앱 버전을 `v1.4.0`으로 갱신했습니다.

## v1.3.0 - 2026-05-26

- 기록 검색, PR 분석, 장비 분석 상단 UI를 대시보드형으로 정리했습니다.
- 기록/PR/장비 화면에 분석 화면 간 이동 액션을 추가했습니다.
- PWA 캐시와 앱 버전을 `v1.3.0`으로 갱신했습니다.

## v1.2.0 - 2026-05-26

- 주간/월간 분석 상단 UI를 연간 리포트와 같은 대시보드 헤드, 액션 버튼, 메트릭 카드 구조로 정리했습니다.
- 분석 화면의 주요 이동 경로를 PR, 운동별, 장비별, 연간 화면과 바로 연결했습니다.
- PWA 캐시와 앱 버전을 `v1.2.0`으로 갱신했습니다.

## v1.1.0 - 2026-05-26

- 오늘 화면의 `기록 상태`를 `분석 신뢰도`로 재정의하고 원형 점수 링, 근거 미니 바, 부족 항목 액션을 추가했습니다.
- 분석 신뢰도 계산을 `health_tracker/services/data_quality.py`로 분리했습니다.
- QA 더미데이터에 체성분 변화와 회복 패턴을 추가해 분석 화면 검증 기반을 강화했습니다.
- PWA 캐시와 앱 버전을 `v1.1.0`으로 갱신했습니다.

## v1.0.0 - 2026-05-26

- 운동 입력 빠른 선택 탭, 컨디션 코칭 카드, 기록 상태 카드 추가
- 기록/분석 페이징, 연도별 분석, QA 더미데이터, QA 리포트 추가

- 전체 QA 회귀 테스트와 PWA 캐시 기준 정비
