# Changelog

## v2.0.1 - 2026-05-26

- 기록 탭 부위별 분석 카드에서 날짜와 운동시간/칼로리 텍스트가 같은 grid 영역에 겹치던 문제를 수정했습니다.

## v2.0.0 - 2026-05-26
