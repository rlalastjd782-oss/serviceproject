# Changelog

이 프로젝트는 Semantic Versioning(`MAJOR.MINOR.PATCH`)을 기준으로 관리합니다.

## v1.9.6 - 2026-05-26

- 오늘 운동 카드에서 같은 운동의 세트명을 한 번에 바꾸는 운동명 일괄 수정 기능을 추가했습니다.
- 운동별 기본 장비 설정값을 오늘 운동 카드와 빠른 운동 선택 버튼에 함께 표시했습니다.
- 세트 수정 폼이 열릴 때 입력칸이 밀리지 않도록 편집 그리드 폭을 정리했습니다.
- PWA 캐시와 앱 버전을 `v1.9.6`으로 갱신했습니다.

## v1.9.5 - 2026-05-26

- 운동 시간 카드 바로 아래에 휴식 타이머를 배치했습니다.
- 장비 카테고리 선택은 장소와 무관하게 기본 카테고리 전체가 항상 보이도록 수정했습니다.
- 장소별 제한은 운동명 후보에만 적용되도록 장비 안내 문구를 정리했습니다.
- PWA 캐시와 앱 버전을 `v1.9.5`로 갱신했습니다.

## v1.9.4 - 2026-05-26

- 오늘 운동 입력의 운동명 후보, 최근 세트, 운동 통계, 즐겨찾기, 루틴 목록을 선택한 운동 장소 기준으로 제한했습니다.
- 장소가 다른 운동명은 선택한 장소의 운동 입력 후보에 섞이지 않도록 했습니다.
- 루틴 저장 시 원본 운동 세션의 장소를 함께 저장하도록 했습니다.
- PWA 캐시와 앱 버전을 `v1.9.4`로 갱신했습니다.

## v1.9.3 - 2026-05-26

- 장비 목록이 운동명이나 장비명으로 오염되지 않도록 장비 값을 카테고리로 정규화했습니다.
- 장소 장비 관리 입력을 자유 텍스트에서 `바벨`, `덤벨`, `머신`, `케이블`, `프리웨이트`, `맨몸`, `유산소기구` 카테고리 선택 방식으로 변경했습니다.
- 기존에 저장된 `스미스 머신`, `런닝머신` 같은 세부 장비명은 각각 `머신`, `유산소기구`로 정리되도록 했습니다.
- PWA 캐시와 앱 버전을 `v1.9.3`으로 갱신했습니다.
